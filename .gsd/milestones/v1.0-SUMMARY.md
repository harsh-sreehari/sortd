## Lessons Learned
- **Jaccard similarity isn't enough**: Path names have too much noise (numbers/dates) for pure Jaccard. Switching to **Keyword Overlap** dramatically improved Tier 2 accuracy.
- **Node runtime churn**: Transitioning to Node 24 in CI requires updating GitHub Actions to newer versions (`v6` for checkout) and setting explicit environment flags for compatibility.
- **Atomic Renames are Risky**: Always ensure the destination is joined with a filename to prevent accidental file-to-folder renaming when directories don't exist yet.
