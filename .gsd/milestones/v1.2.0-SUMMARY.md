## Concerns
- **LLM Context**: The current `review` command calls the LLM for every unknown description. If a user has a massive backlog, this could be slow/expensive.
- **ANSI Color Helpers**: Repeated ANSI color strings in `main.go` (bold, green, red, etc.).

## Recommendations
1. **Batch Review**: Allow reviewing multiple files in one pass with a single LLM prompt for "Bulk Classification" in the future.
2. **UI Lib**: Consider a library for terminal styling (e.g. `charmbracelet/lipgloss`) if the CLI becomes more complex.

## Technical Debt to Address
- [ ] Refactor ANSI color codes into a `ui` or `cli` helper package.
- [ ] Optimize folder crawling for very large home directories (currently recursively walks every time).
